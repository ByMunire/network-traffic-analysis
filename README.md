# Network Traffic Analysis Project

## Overview
This project focuses on analyzing network traffic using Wireshark and basic network security techniques. The goal is to identify normal and suspicious activities within network packets.

## Tools Used
- Wireshark
- Nmap (for traffic generation)
- Basic TCP/IP analysis

## What I Did
- Captured live network traffic using Wireshark
- Applied filters such as DNS, HTTP, TCP, and ICMP
- Analyzed packet details (source IP, destination IP, ports)
- Identified patterns like DNS queries and TCP connections
- Practiced detecting suspicious activities such as port scanning

## Key Learnings
- Understanding of TCP/IP protocols
- Ability to analyze network packets
- Identifying potential security threats in network traffic

## Sample Filters Used
- dns
- http
- tcp.flags.syn == 1
- icmp

## Purpose
This project demonstrates hands-on experience in network traffic analysis and supports skills required for SOC Analyst and Network Security roles.
